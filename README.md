# TO DO Web Application

Vite + React를 이용하여 만든 웹 애플리케이션
- 배포 사이트: https://blumiv-react-todo-app.vercel.app/

## 프로젝트 시작하기

```bash
bun install
bun dev
```

## 프로젝트 빌드하기

```bash
bun install
bun run build
```
